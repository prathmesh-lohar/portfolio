Project Overview

The AI Based Smart Receptionist is the smat receptionist program created by using artificial intelligence and machine learning .

AI Based Smart Receptionist is the complete product that can help to an organizations or institutes by taking the role of virtual receptionist. This project can be works as a Human Receptionist by providing awesome features.

It accepts users queries as a voice command then after process them it gives response to the users by producing speech with the help of output devices. This AI Based Smart Receptionist able to give answers based on training data as well as it can produce answers itself.

This Receptionist Program able to give response bassed on training data and also scrape realtime data from websites for some quries it use libraries like wikipidia .

For training its neuron we provide data sample and its possible answers and run training.py it get update iself and ready to handle new queries.
