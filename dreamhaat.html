DreamHaat – Ecommerce Website

This page contains detailed information about DreamHaat, a fully functional ecommerce website designed for selling organic products and cold-pressed oils online.

Live Link

Project Overview

DreamHaat is a complete ecommerce platform built to provide a smooth and modern shopping experience for organic products and cold-pressed oils. It includes a dynamic admin dashboard that allows easy management of products, categories, and website content.

Users can browse systematically categorized products, place orders, and track them using their mobile number and order ID, making the buying process seamless and user-friendly.

Key Features

  1. 1) Dynamic Product Management: Add, edit, or delete products using an intuitive admin dashboard.
  2. 2) Systematic Categorization: Products are organized into categories for easy browsing.
  3. 3) Order Placement & Tracking: Users can place orders and track them using mobile number + order ID.
  4. 4) Responsive Design: Fully mobile-friendly UI built with Bootstrap.
  5. 5) Dynamic CMS: Complete control over product listings, categories, and website content.
